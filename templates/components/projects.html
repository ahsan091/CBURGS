GuardianEye - Artificial Intelligence Security Operations Center System

A battle-tested Security Operations Center (SOC) platform that unifies the "Big Four" open-source defense tools—Wazuh, TheHive, Shuffle, and OpenCTI—into a single, containerized ecosystem. GuardianEye goes beyond monitoring by integrating directly with Active Directory and pfSense Firewalls to instantly contain threats at the network and identity level.

Core Capabilities

  • › Unified Communcation: All tools configured on a containerized mesh network for seamless API orchestration.
  • › Active Defense: Root-level access to AD & Firewall allowed for automated endpoint isolation and policy enforcement.
  • › AI-Driven Policy: Generative AI analyzes alerts, proposes firewall rules/AD policies, and implements them upon administrator confirmation.

Infrastructure

Built on a dual-container architecture (App + PostgreSQL) with a Django frontend providing a seamless "Wazuh-like" dashboard for centralized control.
